DUMPDATA
Резервное копирование базы данных

1. Использовать базу данных postgre/mysql
2. Применить комманду
python manage.py dumpdata > db.json
3. Появится файл с копией БД
4. При необходимости использовать комманду 
python manage.py loaddata db.json
5. Это поставит резервную копию БД
